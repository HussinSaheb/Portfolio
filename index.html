Smart Pill Dispenser

This project involved using Arduino micro controller to prototype a smart pill dispenser and its functionality. The Smart Pill dispenser was aimed at the elderly with disabilities that require reminders to take their medication. Project was completed using the Programming language Java along side the Arduino IDE and several kits to implement better functionality for the Arduino. MIT App Inventor was used to create a prototype android app for bluetooth communication with the smart pill dispenser

Python - Sentiments analysis

Hussin worked on a python script, that uses the twitter api to retrieve tweets about a certain search term, the retrieved tweets are recorded into a .txt file to serve to IndicoIO API for sentiment analysis. IndicoIO returns with five emotions for each tweet. Hussin then totalled the emotions conveyed behind each tweet and displayed his results in a graph of emoticons representative of the emotion provided by IndicoIO api. The bigger the emoticon icon the higher the emotions were from each tweet.

Raspberry pi/Python script

Athan is a call to prayer that is projected from mosques back in the muslim countries. This project was something Hussin worked on to simulate the environment back home for his family. Hussin wrote a python script run on Raspberry pi, that scrapes the times of prayer every night at 00:10 using cron jobs. The scraped data is processed and used as trigger for the Athan soundfile. To account for the prayer times changing daily, the script is set to quit at 11pm and rerun using the aforementioned cron job to acquire the new prayer times.

TUI

Hussin and two other colleagues worked on a prototype for a car modification game. The game used physical representation of car parts that the game allowed to be modified. depending on how each item was rotated a different version of the part was added to the car in-game. Hussin was responsible for coding the functionality of the game. The game was programmed using Processing (software sketchbook using Java) with the Reactivision framework for computer vision, to track the physical objects.

Passera

Passera is a project Hussin worked on to make a system similar to UBER; instead of hailing taxi's a learner driver would be put in contact with a driving instructor. HTML, CSS , JavaScript, Node.js, MySQL and Apache were used in this project. This was mainly a prototype to test if the idea could be made into fruition. The real project is to make it into an smart phone app for convenience of the target audience. This would also serve as an opportunity for Hussin to gain more experience in mobile app development.
